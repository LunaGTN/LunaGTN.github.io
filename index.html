Mon premier site web

Ici, j'ai pu m'initier au HTML et au CSS pour créer mon premier site web. C'est un site plutôt basique, ce sont mes premiers pas

Les fondamentaux du HTML

HTML n'est pas vraiment un langage de programmation, c'est plutôt du balisage. A vrai dire, il existe un débat dans la communauté informatique. En effet, en HTML on ne va pas déclarer de variable ou appeler de fonctions. Le L de HTML signifie pourtant bien langage non ?

HTML sert a gérer le contenu du site, les informations que l'on veut mettre en avant, la taille des titres etc... Il ne permettre a aucun moment de décorer le site pour le rendre agréable aux yeux de l'utilisateur

Les fondamentaux du css

CSS est l'acronyme de Cascading Style Sheets. Ces fichiers vont décrire la présentation d'une page, c'est-à-dire sous quelle forme le contenu sera affiché.

Ici, on va pouvoir choisir la couleur, la police d'écriture, si on veut mettre le texte en gras ou le surligner. Ce n'est pas non plus un langage de programmation, mais cette fois, il n'y a pas de débat.
